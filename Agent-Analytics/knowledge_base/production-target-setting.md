# Production Target Setting

## Formula
```
Production Target = Available Time × OEE Target × Ideal Cycle Rate
```

## Available Time Calculation
| Shift | Total Minutes | Breaks | Available |
|-------|--------------|--------|-----------|
| Morning (06:00-14:00) | 480 | 30 min | 450 min |
| Afternoon (14:00-22:00) | 480 | 30 min | 450 min |
| Night (22:00-06:00) | 480 | 30 min | 450 min |

## OEE Targets
| Classification | OEE % | Description |
|---------------|-------|-------------|
| World Class | ≥ 85% | Excellent |
| Good | 75-84% | Above average |
| Average | 60-74% | Needs improvement |
| Poor | < 60% | Significant losses |

## Setting Targets
1. Calculate theoretical maximum output
2. Apply OEE target factor (start at 75%)
3. Adjust for known constraints (material, labor, maintenance)
4. Review historical attainment (last 4 weeks average)
5. Target should be achievable — aim for 5-10% above last month's actual

## Monitoring
- Track OEE dashboards daily
- Investigate any target miss > 10%
- Adjust targets monthly based on trend
