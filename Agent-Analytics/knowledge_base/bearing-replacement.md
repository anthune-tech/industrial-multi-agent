# Bearing Replacement Procedure

## Prerequisites
- LOTO applied
- Replacement bearing SKU verified
- Correct tools available (bearing puller, induction heater)
- Work order approved

## Removal
1. Remove shaft guards and coupling
2. Mark positions for reassembly reference
3. Use bearing puller — never hammer directly on bearing
4. Inspect shaft for scoring or wear
5. Clean shaft journal with emery cloth

## Installation
1. Heat new bearing to 80-110°C using induction heater (NOT a torch)
2. Slide bearing onto shaft until seated against shoulder
3. Allow to cool naturally — do not quench
4. Pack bearing with NLGI #2 grease (30% fill)
5. Install seal and bearing cap
6. Reassemble coupling and guards

## Post-Installation Check
- Rotate shaft by hand — should spin freely
- Run at low speed for 5 minutes
- Check temperature every 5 minutes for 30 minutes
- Max acceptable temperature rise: 40°C above ambient
- Record vibration reading for baseline

## Bearing Life
- Expected life: 20,000 - 50,000 hours (load-dependent)
- Replace in pairs (both bearings on same shaft)
- Premature failure investigation required if < 6 months
