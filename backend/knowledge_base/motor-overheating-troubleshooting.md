# Motor Overheating Troubleshooting

## Symptoms
- Motor casing temperature > 90°C
- Thermal overload relay tripped
- Burning smell from motor windings

## Diagnosis Steps
1. Check ambient temperature (should be < 40°C)
2. Verify airflow — clean cooling fan and vents
3. Measure current draw on all 3 phases (balance within 10%)
4. Check for voltage imbalance (should be < 1%)
5. Inspect bearings for wear or insufficient lubrication
6. Verify load is within motor nameplate rating

## Resolution
| Cause | Action |
|-------|--------|
| Blocked ventilation | Clean fan covers and cooling fins |
| Overloaded | Reduce load or upgrade motor |
| Voltage imbalance | Check supply connections and transformer |
| Bearing failure | Replace bearings and regrease |
| Frequent starts/stops | Adjust start frequency or add soft starter |

## When to Escalate
- If temperature exceeds 120°C, shut down immediately
- If smoke or burning smell present, call maintenance supervisor
