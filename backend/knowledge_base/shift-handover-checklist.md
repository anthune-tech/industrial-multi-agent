# Shift Handover Checklist

## Outgoing Shift (Last 15 minutes)
- [ ] Complete all open work orders
- [ ] Log machine status in the system
- [ ] Record OEE data for the shift
- [ ] Note any ongoing issues or anomalies
- [ ] Clean work area to 5S standards
- [ ] Restock consumables if below minimum levels

## Handover Meeting
- [ ] Review production vs target
- [ ] Discuss equipment issues or alarms
- [ ] Highlight safety concerns
- [ ] Share quality defects found
- [ ] Pass along supervisor instructions

## Incoming Shift (First 15 minutes)
- [ ] Attend shift brief with supervisor
- [ ] Review logbook for previous shift
- [ ] Check machine status board
- [ ] Verify material availability for next run
- [ ] Perform 5-minute safety walk
- [ ] Sign off on handover form

## KPIs to Track
- Production output (units/shift)
- OEE percentage
- Scrap/rework percentage
- Safety incidents (0 is the target)
- Downtime minutes and reasons
