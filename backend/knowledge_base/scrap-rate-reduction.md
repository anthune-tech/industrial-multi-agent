# Scrap Rate Reduction

## Current Standard
- Acceptable scrap rate: < 2% of total production
- Target: 1% or less (World Class)
- Measure: Scrap units / Total units produced × 100

## Root Cause Categories
| Category | Typical % | Primary Causes |
|----------|-----------|----------------|
| Process | 45% | Temperature, pressure, speed variations |
| Material | 25% | Incoming material defects, contamination |
| Human | 15% | Setup errors, handling damage |
| Tooling | 10% | Worn tools, incorrect tool selection |
| Other | 5% | Environmental, measurement errors |

## Reduction Methodology
### Step 1: Measure
- Track scrap by machine, shift, product, and defect type
- Use Pareto analysis to identify top 3 defect types

### Step 2: Analyze
- Fishbone diagram for each top defect type
- 5-Why analysis to find root cause
- Identify corrective actions

### Step 3: Improve
- Implement corrective actions
- Update work instructions and SOPs
- Train operators on new procedures

### Step 4: Control
- Monitor scrap rate weekly
- Control charts with upper control limit
- Immediate investigation if UCL exceeded

## Quick Wins
- Verify raw material certifications before use
- First-piece inspection after every setup change
- Tool change at defined intervals (not just when broken)
