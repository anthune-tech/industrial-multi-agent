# Sensor Calibration Procedure

## Applicable Sensors
- Temperature sensors (thermocouples, RTDs)
- Pressure transmitters
- Flow meters
- Position sensors (proximity, photoelectric)
- Vibration sensors

## Calibration Frequency
| Sensor Type | Frequency | Tolerance |
|-------------|-----------|-----------|
| Temperature | Quarterly | ±2°C at range mid-point |
| Pressure | Monthly | ±0.5% of full scale |
| Flow | Semi-annually | ±1% of reading |
| Position | Weekly | ±0.1mm |
| Vibration | Monthly | ±5% of reading |

## Procedure
1. Isolate sensor from process where possible
2. Connect calibrated reference standard
3. Apply known input at 0%, 25%, 50%, 75%, 100% of range
4. Record as-found readings
5. Adjust if outside tolerance
6. Record as-left readings
7. Update calibration sticker with date and technician ID
8. Document in calibration database

## Definition
- **As-found**: Readings before adjustment
- **As-left**: Readings after adjustment
- If as-found is > 2x tolerance, perform root cause analysis

## Failed Sensor Protocol
- Tag sensor as "OUT OF CAL — DO NOT USE"
- Enter work order for replacement
- Review all data generated since last good calibration
